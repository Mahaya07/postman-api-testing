# AI-Assisted API Test Scenario Generation

## Prompt Used
"Generate positive, negative, and edge test cases for a POST API that creates a resource using JSON payload."

## AI-Generated Suggestions (Reviewed)
- Missing Content-Type header
- Invalid JSON structure
- Empty request body
- Incorrect data type for fields
- Unsupported HTTP method

## My QA Review
- Selected relevant scenarios applicable to the API
- Excluded unrealistic cases
- Converted valid suggestions into executable Postman tests

AI output was used as guidance, while final testing decisions were made based on QA judgment.
